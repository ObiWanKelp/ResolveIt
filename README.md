# **Project Lumos – RESOLVE IT**

### *“Report, Resolve, Rebuild – Smarter Campus Living.”*

---

## **Problem Statement**

**Theme:** *Hack the Campus — Innovate for Everyday Life*
**Problem:** Lack of a centralized, transparent system for students to report and track maintenance issues efficiently.

---

## **Our Solution**

**RESOLVE IT** is a simple yet powerful web platform that allows:

* Students to **report maintenance issues** easily
* Staff to **track and update** them
* Admins to **monitor progress** across the campus

## ⚙️ **Features**

### **Student Panel**

* Smart form validation

* Status update controls

## **Database Design (RDBMS Schema)**

**Tables Used:**

Students 
Staff 
Admin 
Issues 


**Relationships:**

* One **Student** can report **many Issues**
* One **Staff** member can handle **many Issues**
* **Admin** has full control and access over all records
